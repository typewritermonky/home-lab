# home-lab
At home lab built on Windows 11 using VirtualBox, made with pentesting in mind. I'm still learning, so whoever stumbles on this feel free to give your 2 cents.

# Environment
- Windows 11 Host OS (ACER)
- Virtualbox Hypervisor
- Target OS: Metasploitable 2 VM
- Attacker OS: Kali Linux
- Network: Host only (isolated)

# Goals
- Learn and exploit vulnerabilities
- Try different things out and build an understanding of networking

# Structure
- /setup --> basic config and notes
- /labs --> writeups and findings
- /screenshots --> pictures and proof
